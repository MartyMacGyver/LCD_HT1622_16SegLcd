# LCD_HT1622_16SegLcd
Experiments with a 9-character 16-segment alphanumeric LCD

Board: ETM8809K2-02

Manufacturer: http://www.canton-electronics.com

Chipset: HT1622 or equivalent

LCD = 9 digit 16 seg + 3 dp + 'HZ' symbol + continuous backlight
